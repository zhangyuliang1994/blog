### SEO 优化的流程我一共分为6个步骤，分别是：

1. 关键词研究与分析
2. 页面内容优化
3. 技术SEO优化
4. 分析工具配置
5. 外链建设策略
6. SEO效果监测

## 1. 关键词研究与分析

### 关键词收集工具

- [SEMRUSH](https://semrush.com/)的Keyword Magic Tool
    - 使用方法和筛选技巧
    - 竞争度分析（KD值解读）
    - 长尾关键词发掘
- [Similarweb](https://similarweb.com/)的关键词生成器
    - 竞品关键词分析
    - 市场趋势洞察
- [Google搜索框自动补全](https://www.google.com/search?q=)
    - 用户搜索意图分析
    - 相关搜索词收集
- [Google Ads关键字规划工具](https://ads.google.com/home/tools/keyword-planner/)
    - 新关键词发现
    - 搜索量和竞争度分析
    - 季节性趋势分析
- [KeywordTool.io](https://keywordtool.io/) 搜索扩展关键词
    - [Google Keyword Tool](https://keywordtool.io/google) Google搜索量分析
    - [YouTube Keyword Tool](https://keywordtool.io/youtube) YouTube搜索量分析
    - [Bing Keyword Tool](https://keywordtool.io/bing) Bing搜索量分析
    - [App Store Keyword Tool](https://keywordtool.io/app-store) 应用推广优化
    - [Play Store Keyword Tool](https://keywordtool.io/google-play) Android应用优化
    - [Instagram Keyword Tool](https://keywordtool.io/instagram) 社交媒体标签
    - [Twitter Keyword Tool](https://keywordtool.io/twitter) 话题标签研究
    - [Pinterest Keyword Tool](https://keywordtool.io/pinterest) 图片内容优化
    - [TikTok Keyword Tool](https://keywordtool.io/tiktok) 短视频内容优化
    - [Google Trends](https://trends.google.com/) 搜索趋势分析

### 关键词筛选标准

- KD值 < 30（可考虑KD < 40的词）
- 月搜索量 > 600-800
- 商业价值评估
- 用户意图匹配度

### 关键词层级规划

- 主关键词确定
- 次关键词布局
- 长尾关键词分布

## 2. 页面内容优化

### 使用Cursor优化内容

- 关键词密度控制（3-5%）
- 内容长度规划（600-800字/页）
- 内容结构优化
- 用户体验提升
