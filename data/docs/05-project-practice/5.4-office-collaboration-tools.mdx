# 5.4 💻 落地页面开发教程

# Landing Page开发

本教程将指导你如何使用 [aimaker-template](https://demo.aimaker.dev/) 中的组件和模板来开发产品落地页。我们以 Creamy Weather Icons 项目为例。

落地页的开发这里主要是将了如何使用 [aimaker-template](https://demo.aimaker.dev/) 中的template和组件模板修改成自己的项目。

模板在这里查看[demo.aimaker.dev](https://demo.aimaker.dev/)

## 开发建议

`@page.tsx 帮我根据PRD文档 @PRD.md 改这个首页的各个组件中的文案，不要改组件的样式，就改组件中的文案。用英文。`

1. 保持组件结构清晰
2. 复用模板提供的组件
3. 遵循设计规范
4. 注意性能优化
5. 做好响应式适配

访问 [http://localhost:3000](http://localhost:3000/) 预览效果。
