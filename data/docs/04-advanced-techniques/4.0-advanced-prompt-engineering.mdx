# 4.0 🤖 全栈开发指南-概览

# AI 开发入门指南

这个章节讲的都是AI开发常用的技术栈，在了解了这些技术栈之后，你就可以开始使用AI开发复杂的产品了。

## 基础知识

在开始 AI 开发之前，你需要了解一些基础知识：

### [网站基础 HTML/CSS/JS](https://new.web.cafe/tutorial/detail/st7lswuwih)

作为 AI 开发者，你不需要精通前端开发，但需要了解基本的 HTML 和 CSS 概念，以便能与前端开发者进行有效沟通。

### [认识 Markdown](https://new.web.cafe/tutorial/detail/71k484wemi)

Markdown 是一种轻量级标记语言，常用于编写文档。AI 输出的内容通常使用 Markdown 格式，我们也用它来写 PRD 文档和 GitHub README。

### [认识终端](https://new.web.cafe/tutorial/detail/n8e6k068y5)

终端是开发者的重要工具，你需要了解基本的命令行操作。不用担心记不住所有命令，用到时询问 AI 即可。

### [网站类型](https://new.web.cafe/tutorial/detail/q45qjfqs56)

了解不同类型的网站，知道什么样的网站我们能做，什么样的网站我们不能做，这对项目规划很重要。

## 开发工具

我们使用以下工具进行开发：

### [认识 GitHub](https://new.web.cafe/tutorial/detail/f4sknyvk76)

GitHub 是代码托管平台，我们用它进行版本控制和团队协作。你需要了解基本的 Git 操作。

### [认识 Next.js](https://new.web.cafe/tutorial/detail/k0h87ghqab)

Next.js 是我们使用的 React 框架，它提供了完整的全栈开发解决方案，让开发变得更简单。

### [认识 Tailwind CSS](https://new.web.cafe/tutorial/detail/lzi1tbiofj)

Tailwind CSS 是一个实用的 CSS 框架，通过预设的类名快速构建界面样式。

### [认识 Shadcn UI](https://new.web.cafe/tutorial/detail/p0gjrrq22b)

Shadcn UI 提供了一系列精美的 UI 组件，帮助我们快速构建漂亮的界面。

## 部署服务

项目开发完成后，需要部署到服务器上：

### [认识 Vercel](https://new.web.cafe/tutorial/detail/8vm5cxtk7n)

Vercel 是我们的主要部署平台，它提供了简单的部署流程和强大的功能。

### [认识 Cloudflare](https://new.web.cafe/tutorial/detail/yv7x0z7gmk)

Cloudflare 提供了 CDN、域名管理、R2 存储等服务，帮助我们优化网站性能。

## 数据存储

### [认识数据库](https://new.web.cafe/tutorial/detail/2vd1j6392t)

了解数据库的基本概念，知道如何选择合适的数据库解决方案。

## 学习路线

1. 首先了解 [网站类型](https://new.web.cafe/tutorial/detail/q45qjfqs56) 和 [HTML/CSS 基础](https://new.web.cafe/tutorial/detail/st7lswuwih)
2. 学习 [Markdown](https://new.web.cafe/tutorial/detail/71k484wemi) 和 [终端](https://new.web.cafe/tutorial/detail/n8e6k068y5) 的基本使用
3. 注册 [GitHub](https://new.web.cafe/tutorial/detail/f4sknyvk76) 账号，了解基本操作
4. 学习 [Next.js](https://new.web.cafe/tutorial/detail/k0h87ghqab)、[Tailwind CSS](https://new.web.cafe/tutorial/detail/lzi1tbiofj) 和 [Shadcn UI](https://new.web.cafe/tutorial/detail/p0gjrrq22b)
5. 了解 [Vercel](https://new.web.cafe/tutorial/detail/8vm5cxtk7n) 和 [Cloudflare](https://new.web.cafe/tutorial/detail/yv7x0z7gmk) 的部署流程
6. 最后学习 [数据库](https://new.web.cafe/tutorial/detail/2vd1j6392t) 的基础知识

记住，你不需要一次性掌握所有内容。跟着教程一步步来，遇到问题时及时询问 AI，慢慢就会熟悉起来。

祝你学习愉快！🚀
