# 2.0 🛠️ 软件工具教程

# 概览

在开发过程中，我们需要使用各种工具来提高效率和质量。以下是推荐的核心软件工具：

## 开发工具

- **GitHub Desktop** - 图形化的 Git 客户端，让版本控制更简单直观
- **Raycast** - 强大的效率启动器，可以快速访问应用和执行命令

## 设计工具

- **Figma** - 专业的 UI 设计工具，支持协作和原型设计

## AI 辅助工具

- **Glarity** - 智能 AI 助手，提供编程和其他任务的帮助
- **Bob** - 便捷的翻译和 OCR 工具，支持多种语言

## 网站工具

- **SiteSucker** - 网站下载工具，可以保存完整网站内容
- **Easy Scraper** - 简单易用的网页数据采集工具
- **Lighthouse** - 网站性能分析工具，提供优化建议

这些工具各自专注于不同的领域，配合使用可以显著提升开发效率。点击各个工具的详细页面可以了解具体的安装和使用方法。
