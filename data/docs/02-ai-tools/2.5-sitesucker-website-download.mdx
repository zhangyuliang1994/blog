# 2.5 📥 SiteSucker: 网站下载

# SiteSucker

官方网站：https://ricks-apps.com/osx/sitesucker/

**SiteSucker 是一个专业的网站下载工具**，它不仅能下载网站的 HTML 源文件，还能完整保存网站的整体架构、文本、程序文件、图片、视频、音频等所有数据到本地。使用非常简单，只需输入网站 URL 并点击开始即可。

## 核心功能

SiteSucker 提供了强大的网站下载功能：

- **完整下载**：保存网站的所有页面和资源，包括 HTML 文档、图片、背景、动画等
- **结构保持**：完整复制网站的目录结构和链接关系
- **自动跟踪**：智能跟踪和下载相关链接
- **过滤规则**：自定义下载内容的范围和类型
- **断点续传**：支持中断后继续下载
- **离线浏览**：下载后可完整离线访问
- **进度监控**：实时查看下载文件数量、错误数量等状态
- **历史记录**：查看最近下载的网站列表
- **队列管理**：支持显示/隐藏下载队列

## 使用特点

- **简洁界面**：直观的操作界面，可轻松启动、暂停或停止下载
- **日志查看**：支持检查下载日志，方便排查问题
- **文件管理**：可打开和管理下载的文件和文件夹
- **灵活配置**：可选择是否对下载文件进行本地化处理
- **选择性下载**：支持跳过不需要的文件

SiteSucker 让网站内容的本地化变得简单高效，是网站备份和学习的理想工具。
