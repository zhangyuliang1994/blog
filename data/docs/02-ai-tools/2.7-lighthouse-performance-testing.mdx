# 2.7 📊 Lighthouse: 性能检测

# Lighthouse

官方网站：https://chromewebstore.google.com/detail/blipmdconlkpinefehnmjammfjpmpbjk

**Lighthouse 是 Google 开发的网站性能检测工具**，它能全面分析网站的性能、可访问性、最佳实践和 SEO 表现，就像是给网站做了一次全面体检。

## 核心功能

Lighthouse 提供了全面的网站检测功能：

- **性能分析**：测试加载速度、交互延迟、渲染性能
- **可访问性**：检查网站对残障用户的友好程度
- **最佳实践**：评估代码质量和开发规范
- **SEO 检查**：分析搜索引擎优化程度
- **PWA 支持**：检测渐进式 Web 应用特性
- **详细报告**：提供具体的优化建议

Lighthouse 让网站优化变得有据可依，帮助开发者构建更好的 Web 应用。
