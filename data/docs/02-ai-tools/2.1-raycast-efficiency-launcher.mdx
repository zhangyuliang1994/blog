# 2.1 🚀 Raycast: 效率启动器

# Raycast

官方网站：https://www.raycast.com/

## 为什么要用 Raycast？

Raycast 是一个必备的快速启动器应用，专门给 Mac 用户设计，Raycast 涵盖了 Spotlight 的所有功能，能够迅速启动应用和文件，执行各种复杂操作，应用切换、文件搜索和系统控制等任务，还有丰富的插件系统，可以扩展更多功能。

## 核心功能

在 Raycast 的世界里，一切操作都变得简单高效：

- **快速启动**：瞬间打开任何应用
- **剪贴板历史**：不再担心复制内容的丢失
- **智能计算**：直接进行各种运算
- **插件系统**：扩展更多功能
- **窗口管理**：整理凌乱的桌面
- **AI 助手**：智能对话和任务处理（Pro 版）
- **云同步**：跨设备同步设置（Pro 版）

### 剪贴板历史记录

Raycast 的剪贴板管理是最实用的功能之一，可以帮你记录和管理所有复制过的内容：

- **快速访问**：使用快捷键 `⌘` `⇧` `V` 快速打开剪贴板历史
- **智能预览**：支持文本、图片、文件等多种格式的内容预览
- **快捷搜索**：可以在历史记录中搜索关键词
- **固定收藏**：常用内容可以固定在顶部（使用 `⌘` `P` 快捷键）

使用技巧：

1. 使用 `↑` `↓` 键在历史记录中导航
2. 按 `Enter` 直接粘贴选中的内容
3. 右键点击可以查看更多操作选项
4. 在设置中可以调整历史记录保存的数量和时间

### 代码片段（Snippets）

Snippets 是 Raycast 中非常实用的文本模板功能，可以帮你快速输入常用的文本内容：

### 基本用法

- 打开 Snippets 管理
- 新建 Snippet 时可以设置名称、内容和触发关键词
- 在任意输入框中输入触发关键词，就能快速插入预设内容

### 设置快捷键

Raycast 搜索 Snippets 会出来Snippets的设置/添加/搜索/编辑选项

![](https://s.web.cafe/aimaker/images/software/raycast/snippets_edit.webp)

### 示例一：

在任何输入框输入 `#cmd` 会出来 `⌘` 符号

![](https://s.web.cafe/aimaker/images/software/raycast/snippets.webp)

### 示例二：文章总结模板

```
触发关键词: #sum 内容: 你是一个文章阅读助手，帮助我总结文章并进行必要的批判性思考。请：
用100字简要阐述文章内容 提炼文章最重要的3-5个观点，并分析作者的事实依据与逻辑推演 提炼文章最关键的3-5个数据（如有） 最后用几句话进行整体总结。
```

## 推荐插件

### Zipic - 图片压缩工具

[Zipic](https://zipic.5km.tech/) 是一款高效的图片压缩工具，通过 Raycast 插件可以实现快速的图片压缩。它支持多种格式的图片压缩，并且所有处理都在本地完成，保护用户隐私。

![](https://s.web.cafe/aimaker/images/software/raycast/zipic_screenshot.webp)

### TinyPNG - 图片压缩工具

[TinyPNG](https://tinypng.com/) 是一个流行的在线图片压缩工具，Raycast 插件可以让用户直接在 Finder 中选择图片进行压缩，操作简单高效。

![](https://s.web.cafe/aimaker/images/software/raycast/tinypng_screenshot.webp)

### Unsplash - 图片搜索工具

[Unsplash](https://www.raycast.com/eggsy/unsplash) 是一个强大的图片搜索插件，允许用户在 Raycast 中搜索和下载 Unsplash 上的图片。

![](https://s.web.cafe/aimaker/images/software/raycast/unsplash_screenshot.webp)

### GIF Search - 动态图片搜索工具

[GIF Search](https://www.raycast.com/josephschmitt/gif-search) 插件可以让用户在多个平台上搜索动态 GIF，提升工作效率。

![](https://s.web.cafe/aimaker/images/software/raycast/gif_search_screenshot.webp)

### Kill Process - 进程管理工具

[Kill Process](https://www.raycast.com/rolandleth/kill-process) 插件可以根据 CPU 或内存使用情况终止进程，帮助用户管理系统资源。

![](https://s.web.cafe/aimaker/images/software/raycast/kill_process_screenshot.webp)

### Color Picker - 颜色选择工具

[Color Picker](https://www.raycast.com/thomas/color-picker) 插件可以让用户在 Mac 上选择和组织颜色，适用于设计工作。

![](https://s.web.cafe/aimaker/images/software/raycast/color_picker_screenshot.webp)

### YouTube Downloader - 视频下载工具

[YouTube Downloader](https://www.raycast.com/vimtor/youtube-downloader) 插件可以让用户在 Raycast 中下载 YouTube 视频，方便离线观看。

![](https://s.web.cafe/aimaker/images/software/raycast/youtube_downloader_screenshot.webp)
