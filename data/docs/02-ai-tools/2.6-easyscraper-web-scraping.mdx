# 2.6 🔍 Easy Scraper: 网页采集

# Easy Scraper

官方网站：https://chromewebstore.google.com/detail/cljbfnedccphacfneigoegkiieckjndh

**Easy Scraper 是一个简单易用的网页数据采集工具**，它能帮你轻松获取网页上的各种信息，无需编写任何代码。

## 核心功能

Easy Scraper 提供了强大的数据采集功能：

- **智能选择**：点击选择想要的内容，自动识别相似元素
- **批量采集**：一键获取所有相似数据
- **数据导出**：支持 CSV、Excel 等多种格式导出
- **自动翻页**：支持自动采集多个页面的数据
- **选择器编辑**：可以手动调整选择规则
- **数据预览**：实时查看采集结果

Easy Scraper 让数据采集变得简单直观，是市场调研和数据分析的得力助手。
