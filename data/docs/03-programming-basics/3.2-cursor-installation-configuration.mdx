## 常用快捷键

以下是一些常用的快捷键:

- `Cmd+Shift+P` (Windows/Linux: `Ctrl+Shift+P`): 打开命令面板
- `Cmd+P` (Windows/Linux: `Ctrl+P`): 快速打开文件
- `Cmd+Shift+F` (Windows/Linux: `Ctrl+Shift+F`): 全局搜索
- `Cmd+F` (Windows/Linux: `Ctrl+F`): 当前文件搜索
- `Cmd+J` (Windows/Linux: `Ctrl+J`): 切换终端
- `Cmd+,` (Windows/Linux: `Ctrl+,`): 打开设置

## 安装历史版本

如果您需要安装特定版本的 Cursor，可以访问 [Cursor 历史版本下载页面](https://github.com/oslook/cursor-ai-downloads?tab=readme-ov-file#all-version-download-links)。

在该页面中，您可以：

- 查看所有历史版本的下载链接
- 选择适合您操作系统的安装包
- 下载并安装您需要的特定版本

注意：建议尽可能使用最新版本，除非您有特殊需求必须使用历史版本。

## 推荐扩展

以下是一些推荐的扩展插件:

### 主题美化

- [**Chinese (Simplified) Language Pack**](https://marketplace.cursorapi.com/items?itemName=MS-CEINTL.vscode-language-pack-zh-hans): VS Code 的中文（简体）语言包，提供本地化的界面体验
- [**Ayu Mirage Zed**](https://marketplace.cursorapi.com/items?itemName=enhancedjax.vscode-ayu-zed): 一个基于 Zed 编辑器的 Ayu Mirage 主题，包含文件图标和自定义 CSS

### 开发工具

- [**Color Highlight**](https://marketplace.cursorapi.com/items?itemName=naumovs.color-highlight): 在编辑器中高亮显示颜色代码
- [**Live Server**](https://marketplace.cursorapi.com/items?itemName=ritwickdey.LiveServer): 启动本地开发服务器，支持静态和动态页面的实时重载
- [**Tailwind CSS IntelliSense**](https://marketplace.cursorapi.com/items?itemName=bradlc.vscode-tailwindcss): Tailwind CSS 的智能提示和自动补全

### Markdown/MDX

- [**MDX**](https://marketplace.cursorapi.com/items?itemName=unifiedjs.vscode-mdx): MDX 文件的语法高亮和语言支持
- [**MDX Preview**](https://marketplace.cursorapi.com/items?itemName=xyc.vscode-mdx-preview): MDX 文件的实时预览功能

这些扩展可以通过 Cursor 的扩展市场进行安装。在命令面板中输入 "Install Extension" 或点击左侧活动栏的扩展图标即可访问。

## 如何购买Cursor

Cursor 是一款订阅制的软件，新用户有150次的免费试用。之后需要每个月付费使用，但是$20的价格每月只有500次快速高速请求，超过500之后速度就会很慢，所以我不建议买。

网上有很多免费白嫖的教程，大家可以去搜索折腾，但是白嫖也是有时间成本的，方法一直在变，我不建议花太多时间折腾这个。

我暂时建议大家直接在淘宝搜索'Cursor'找一家店铺去购买，价格也不贵，几十块钱RMB左右一个月但是可以无限次使用高速请求。

注意：这个Cursor的费用是省不了的，因为我们会一直用到，所以建议大家还是在淘宝买一个，有问题淘宝客服也能给你解决，不要花太多时间在这里。

除了Cursor之外也有很多AI辅助编程的工具，但是我都不推荐，因为目前为止效果最好的还是Cursor。
