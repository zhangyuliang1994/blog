# 3.6 🤖 Cursor AI Agent(⌘ I)

# 代理人 (Agent/Composer)

## 基本功能

Cursor Agent/Composer 是一个强大的 AI 编辑工具，可以帮你自动检索上下文，同时修改多个相关文件，创建文件，执行命令，修改错误。

## 使用方法

### 打开 Composer

快捷键: `⌘ + I` 位置: Cursor右上角

### 使用方法

直接说你想做什么。

### 推荐的使用场景

### 1. 新项目创建

比如，你想要创建一个新项目，就可以直接说你想创建一个什么项目，这个时候一般是你已经很清楚你要做什么了，这个时候你就可以直接说你想创建一个什么项目，然后 Cursor Agent/Composer 就会根据你的需求自动创建一个新项目。

### 2. 项目添加某个具体的功能，可能会涉及到多个文件的修改

比如，你想要添加一个功能，但是又不知道具体涉及到哪些文件的修改，就可以直接说你想添加哪个功能。

### 3. 明确某个文件的修改

比如，你想要修改一个文件，但是又不想全局搜索，就可以直接说你想修改哪个文件。
