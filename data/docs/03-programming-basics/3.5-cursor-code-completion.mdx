# 3.5 ⌨️ Cursor代码补全(Tab)

# 智能代码补全 (Tab)

## 基本功能

Cursor Tab 是内置的智能代码补全功能，比 GitHub Copilot 更强大，可以在光标周围进行编辑，而不仅仅是插入代码

## 使用方法

### 基础操作

- 接受建议: 按 `Tab` 键
- 拒绝建议: 按 `Esc` 键

## 代码示例

### 1. 文本翻译

```
// 输入 "前几个字母" 后按 Tab 
transl

const translations = {
  welcome: '欢迎使用',
  settings: '设置',
  profile: '个人资料',
  logout: '退出登录'
}
```

### 2. Tailwind 按钮组件

```
// 输入 "tailwindcss 样式" 后按 Tab
const Button = ({ children, variant = 'primary' }) => {
  const styles = {
    primary: 'bg-blue-500 hover:bg-blue-600 text-white',
    secondary: 'bg-gray-500 hover:bg-gray-600 text-white',
    outline: 'border-2 border-blue-500 text-blue-500 hover:bg-blue-50'
  }
  
  return (
    <button className={`px-4 py-2 rounded-lg transition-colors ${styles[variant]}`}>
      {children}
    </button>
  )
}
```
