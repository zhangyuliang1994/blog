## Mac 软件安装指南

Mac 软件通常有以下几种安装方式：

| 安装方式 | 步骤 | 优点 | 注意事项 |
| :--- | :--- | :--- | :--- |
| App Store 安装 | 1. 在 App Store 中搜索<br/>2. 点击"获取"即可 | - 最安全的安装方式<br/>- 支持自动更新 | - 部分软件可能收费<br/>- 版本可能不是最新 |
| DMG 安装包 | 1. 下载 .dmg 文件<br/>2. 双击打开 DMG 文件<br/>3. 将应用程序拖拽到 Applications 文件夹<br/>4. 弹出安装镜像<br/>5. 删除下载的 DMG 文件 | - 安装过程简单<br/>- 下载速度快 | - 注意下载源的安全性<br/>- 需要手动更新 |
| PKG 安装包 | 1. 下载 .pkg 文件<br/>2. 双击打开安装包<br/>3. 按照安装向导完成安装<br/>4. 删除安装包文件 | - 可自定义安装选项<br/>- 支持系统级安装 | - 可能需要管理员权限<br/>- 卸载相对复杂 |
| ZIP 压缩包 | 1. 下载 .zip 文件<br/>2. 双击解压文件<br/>3. 将解压出的应用移动到 Applications 文件夹<br/>4. 删除压缩包文件 | - 文件体积小<br/>- 便于分发 | - 可能需要手动授权<br/>- 需要手动更新 |
| Homebrew 安装 | 1. 打开终端<br/>2. 运行 `brew install 软件名`<br/>3. 等待安装完成 | - 命令行一键安装<br/>- 自动管理依赖<br/>- 方便更新和卸载<br/>- 支持批量安装 | - 需要先安装 Homebrew<br/>- 部分软件可能版本较旧<br/>- 安装速度取决于网络 |

### 常见问题解决

**Q: 提示"无法打开，因为它来自身份不明的开发者"怎么办？**

A: 打开"系统偏好设置" > "安全性与隐私"，点击"通用"标签，点击左下角的锁图标并输入密码，选择"仍要打开"或"始终允许"。注意：仅对可信任的应用执行此操作。

**Q: 提示"已损坏，无法打开"怎么解决？**

A: 打开终端，输入命令：`xattr -cr /Applications/应用名称.app`，然后重新打开应用。这种情况通常是由系统安全设置导致的。

**Q: 安装时提示"磁盘空间不足"该如何处理？**

A: 检查并清理磁盘空间，可以使用 CleanMyMac 等工具清理系统，删除不需要的大文件或应用。建议保持至少 10% 的磁盘空闲空间。
