### 触控板使用

- **单指点按**: 相当于鼠标左键点击
- **双指点按**: 呼出右键菜单
- **双指滑动**: 上下左右滚动页面
- **双指捏合**: 缩放页面或图片
- **三指向上滑动**: 显示所有窗口
- **五指捏合**: 显示Launchpad

## 系统功能

### Spotlight 搜索

这是 Mac 最强大的功能之一：

- **快速启动应用**: 输入应用名称即可快速启动
- **搜索文件和文档**: 快速定位系统中的任何文件
- **进行简单计算**: 直接在搜索框中输入算式
- **压缩图片**: 搜索"压缩图片"可快速处理图片
- **查找系统设置**: 快速访问系统偏好设置

### 多桌面管理

- **Control + 上箭头**: 进入 Mission Control
- **Control + 左箭头/右箭头**: 切换桌面
- **Control + 数字键**: 切换到指定桌面
- **四指向上滑动**: 显示所有窗口（触控板）
- **四指向左/右滑动**: 切换桌面（触控板）

### 窗口管理

- **Command + H**: 隐藏当前应用
- **Command + Option + H**: 隐藏其他应用
- **Command + M**: 最小化窗口
- **Command + Option + M**: 最小化所有窗口
- **绿色按钮**: 进入全屏模式
- **长按绿色按钮**: 分屏选项

### 效率提升建议

1. **养成使用快捷键的习惯**
   - 刻意练习，减少鼠标使用
   - 先掌握高频快捷键
   - 逐步学习更多快捷键

2. **自定义快捷键**
   - 系统偏好设置 > 键盘 > 快捷键
   - 为常用应用设置自定义快捷键
   - 避免快捷键冲突

3. **使用第三方工具**
   - **Rectangle**: 窗口管理工具
   - **Alfred**: 增强版 Spotlight
   - **Raycast**: 现代化的启动器
   - **Karabiner-Elements**: 键盘映射工具

4. **学习应用特定快捷键**
   - 每个应用都有独特的快捷键
   - 查看应用菜单中的快捷键提示
   - 参考官方文档和教程

## 常见问题解决

### 快捷键不生效

1. 检查系统偏好设置中的快捷键配置
2. 确认没有被其他应用占用
3. 重启应用或系统
4. 检查键盘是否正常工作

### 提高学习效率

1. **循序渐进**: 不要一次性学习所有快捷键
2. **重复练习**: 刻意在工作中使用新学的快捷键
3. **制作备忘录**: 将常用快捷键整理成表格
4. **分类学习**: 按应用或功能分类学习

掌握这些 Mac 快捷键后，您的工作效率将得到显著提升。建议从最常用的快捷键开始学习，逐步扩展到更多的操作技巧。记住，熟练使用快捷键是成为高效开发者的重要技能之一。
