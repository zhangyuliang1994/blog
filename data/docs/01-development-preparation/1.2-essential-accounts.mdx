### 关于科学上网

科学上网是一种访问受限网站的解决方案，通常也被称为：科学上网、魔法上网、梯子、代理、机场等。在选择科学上网服务时,需要权衡以下三个核心要素:

1. **稳定性** - 服务的可靠程度和连接稳定性
2. **速度** - 网络访问速度和延迟情况
3. **便宜** - 价格的经济实惠程度

这三个要素之间存在一些必然的关系:

- 稳定 + 速度 = 贵
- 速度 + 便宜 = 不稳定
- 稳定 + 便宜 = 速度差
- 三者都想要 = 容易跑路

因此在选择时要根据自己的实际需求来权衡,建议优先考虑稳定性,其次是速度,最后才是价格因素。

## OpenAI/Claude API 申请说明

在开发 AI 应用时，我们需要使用 OpenAI 或 Claude 的 API 服务。然而，由于地区限制和申请条件的限制，直接申请官方 API 可能会遇到一些困难。

### 官方 API 申请条件

申请 OpenAI 官方 API 需要满足以下条件：

- 能够访问国际互联网（注意：在未支持地区使用可能导致账号被封）
- 海外手机号（用于接收验证短信）
- 海外信用卡
